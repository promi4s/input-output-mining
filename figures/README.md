Most figures are SVGs created by running our tool with the configuration described in the thesis (such as filtering parameters) on the logs that can be found in `data/evaluation/` in the main branch.
In addition, parameters such as using weighted or variable arcs, or the direction top-to-bottom or left-to-right must be set.


For some figures, I used Inkscape to add further explanations or combined two output figures into one.


In `Implementation/`, I used [edotor.net](https://edotor.net/) to generate the SVG from the graphviz code given in the text files.


The figures in `Runtime/` were generated using `data/evaluation/runtime.py` on the main branch.


The figures in `Plugin/` are modified screenhots from the plugin.


The figures in `Flowchart/` were created using [whimsical.com](https://whimsical.com/) and are not editable without their tool.


Finally, note that the figures from `Discussion/` did not make it into the final version of the thesis.
